# ROOC Auction Bot

Discord Reservation Bot with Web Dashboard.

## 🚀 Deployment
This project is configured for **Hugging Face Spaces** using Docker.

- **URL**: https://huggingface.co/spaces/roocaution/rooc-auction-bot
- **Database**: Supabase (PostgreSQL)

## 🛠️ Management
Use the following commands for updates:
- `npm run ai-update` or double-click `ai_update.bat`
