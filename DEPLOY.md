### 1. วิธีอัปเดตโค้ด (หลังจากแก้ไขงานเสร็จ)
ให้รันคำสั่งตามลำดับนี้ใน Terminal:

```bash
# 1. เตรียมไฟล์และบันทึก พร้อมส่งขึ้น GitHub ทันที
npm run ai-update
```
*ระบบจะส่งโค้ดขึ้น GitHub และ Render จะดึงไปอัปเดตให้อัตโนมัติครับ*

---

### 💡 คำสั่งอื่นๆ ที่ควรรู้
*   `npm run update`: อัปเดตงานแบบรวบตึง (ชื่อบันทึกจะเป็น "Update")
*   `npm run push-github`: ส่งขึ้น GitHub อย่างเดียว (กรณี commit ไว้แล้ว)
